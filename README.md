# JavaScript Quiz App

An interactive quiz application built using vanilla JavaScript.

## Features
- Start screen with username input
- Randomized quiz questions
- Timer-based questions
- Score tracking
- Visual feedback for correct and wrong answers
- Clean and simple UI

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## What I Learned
- DOM manipulation
- Event handling
- State management in JavaScript
- Timer and asynchronous logic
- Debugging and improving UI behavior

## Project Purpose
This project was created to strengthen JavaScript logic, understand async behavior, and practice building interactive web applications without using any frameworks.

## How to Run
- Clone or download the repository
- Open `index.html` in your browser
